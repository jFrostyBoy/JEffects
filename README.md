## JEffects

**Ядро**: Paper / Spigot  
**Версия**: 1.16.5 - 1.21.11  
**Java**: 16+  

### Плагин позволяет накладывать на игроков эффекты зелий, которые:
- **Не исчезают** при смерти игрока
- **Не снимаются** ведром молока
- **Сохраняются** после выхода и входа на сервер
- Таймер длительности эффекта идёт **только пока игрок онлайн** (офлайн — таймер стоит на паузе)
- Настройка дефолтных эффектов, которые выдаются при первом входе на сервер

_Все данные эффектов автоматически сохраняются в `plugins/JEffects/effects.yml`._

### Права доступа
| Право               | Описание                                      | По умолчанию |
|---------------------|-----------------------------------------------|--------------|
| `jeffects.admin`    | Доступ ко всем командам плагина               | Только OP    |

## Команды
**Основная команда**: `/je`

| Команда                                               | Описание                                                                          | Необходимое право |
|-------------------------------------------------------|-----------------------------------------------------------------------------------|-------------------|
| `/je give <игрок> <эффект> <длительность> <уровень>`  | Накладывает постоянный эффект на игрока                                           | `jeffects.admin`  |
| `/je clear <игрок> <эффект>`                          | Снимает конкретный постоянный эффект с игрока                                     | `jeffects.admin`  |
| `/je clear <игрок> all`                               | Снимает **все** постоянные эффекты с игрока                                       | `jeffects.admin`  |
| `/je clear all all`                                   | Снимает **все** постоянные эффекты со всех игроков                                | `jeffects.admin`  |
| `/je info <игрок>`                                    | Показывает список всех активных постоянных эффектов игрока с оставшимся временем  | `jeffects.admin`  |
| `/je reload`                                          | Перезагружает плагин. Эффекты игроков остаются без изменений                      | `jeffects.admin`  |

### Подробности по команде `/je give`
- `<игрок>` — ник игрока (можно указывать оффлайн-игрока)
- `<эффект>` — название эффекта на английском (регистр не важен).  
  Примеры: `speed`, `jump_boost`, `night_vision`, `fire_resistance`, `regeneration`, `strength` и т.д.
- `<длительность>` — время действия в формате:
    - `30s` — 30 секунд
    - `15m` — 15 минут
    - `2h` — 2 часа
    - `7d` — 7 дней
    - `2w` — 2 недели
    - `-1` — бесконечно
- `<уровень>` — уровень эффекта (1, 2, 3…)

**Пример:**  
- `/je give Notch speed 2h 3` — даёт игроку Notch эффект Скорость III на 2 часа.

## Особенности использования
- Игроку **не отправляются** никакие сообщения о наложении или снятии эффектов.
- Все сообщения (успех, ошибки и т.д.) видит **только администратор**, выполнивший команду.
- Все команды имеют **полное таб-дополнение** (ники игроков, названия эффектов, примеры длительности, уровни, `all`).
- Настройки сообщений можно изменить в файле `plugins/JEffects/config.yml`.

## Файлы плагина
- `config.yml` — настройки сообщений
- `effects.yml` — хранилище всех активных постоянных эффектов (не редактируйте вручную!)
